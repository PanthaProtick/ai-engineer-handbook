# Explain the difference between the Internet and the Web.
Internet is an infrastructure that connects devices all over the globe. The web is an internet service which is associated with websites and browsers.
# Why do computers use IP addresses instead of domain names?
It's easier and space efficient for computer to use numbers instead of strings.
# Why does DNS exist?
To convert human readable domain names to IP addresses by which the browsers can interact with the server. 
# What problem do ports solve?
Ports isolates each process running in a computer or a device. 
# What does localhost:8000 mean?
localhost represents IP of current computer and 8000 means the port that is exposed for this process.
# Write the complete journey from typing google.com until the webpage appears.
1. Browser searches locally for IP addresses in the computer
2. If not then ask DNS servers associated with the ISPs
3. DNS server sends back the IP address
4. After recieving the IP browser sends the GET request to the server for the webpage
5. Server sends a response with a header and the html file alongside other resources like js, css, image files etc.
6. Browser assembles it and shows it to the user.
# Break this URL into its components:
## https://example.com:8000/api/users?id=5
Scheme: https
Authority: example.com:8000
Path to resources: api/users
Parameter: id=5 (key, value)
